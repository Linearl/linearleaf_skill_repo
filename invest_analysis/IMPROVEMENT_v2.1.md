## 二、新增Step：商品周期分析（Step 2.5）

### 2.5.1 商品定价周期的历史复盘

**目的：** 理解当前商品价格处于历史周期的什么位置

**执行方法：**
```markdown
### 第一步：收集历史价格数据（过去5-10年）

对于AI芯片，关键价格指标：
- 高端芯片代工价格 (台积电/三星报价指数)
- 通用芯片平均售价 (均价ASP，按工艺分类)
- 原材料成本指数 (晶圆成本)

### 第二步：划分周期四阶段

根据价格变化，识别：
1. 极度低估阶段（底部）
   特征：价格触及成本线附近
   持续：1-2年

2. 缓慢修复阶段（上升初期）
   特征：价格从成本线向正常价缓慢上升
   持续：2-3年
   ← 【当前AI芯片可能处于此阶段】

3. 价格狂热阶段（上升末期）
   特征：价格远超历史中位数，市场乐观情绪高涨
   持续：1-2年

4. 再次崩塌阶段（下降期）
   特征：供给增加，需求见顶，价格快速下跌
   持续：1-3年

### 第三步：标注历史转折点

标记：
- 产能大幅扩张的时间点
- 新技术应用的启动时间
- 政策变化的关键节点
- 地缘冲突的影响时间

### 第四步：当前位置定位

在四阶段模型中，确定：
- 我们在第几阶段？
- 距离下一个阶段还有多远？
- 这个阶段的时间长度预期？
```

**输出格式：**
```
周期阶段坐标表：

时间         |  2020年  |  2021年  |  2022年  |  2023年  |  2024年  |  2025年末 |
─────────────┼──────────┼──────────┼──────────┼──────────┼──────────┼────────────┤
价格(相对)   |   50%    |   70%    |   85%    |   95%    |  110%    |   120%  ← 当前
周期阶段     |   1→2    |    2     |    2     |   2→3    |    3     |    3    
里程碑       |  底部    |  缓慢涨  |  持续涨  |  加速涨  | 高位平台 | 顶部试探?
风险指标     |  ✓低位   |  ✓中低   |  🟡中等   |  🟡中高   |  🔴高位  | 🔴顶部?
```

### 2.5.2 供需基本面分析

**目的：** 用五个关键指标判断周期何时翻转

**五个核心指标：**

#### 指标1：产能分析（供给弹性）
```
定义：全球在线产能 + 在建产能 + 规划产能

对于AI芯片：
- TSMC AI芯片产能：当前月产 XXX万片 (来源：业绩说明会)
- 预计2025年新增产能：+30% (扩建中)
- 预计2026年新增产能：+20% (规划中)
- 三星、Intel的AI产能：另外+50% (长期威胁)

判断标准：
- 如果新产能投产速度 < 需求增速 → 产能紧张继续（利好）
- 如果新产能投产速度 ≈ 需求增速 → 保持平衡（中性）
- 如果新产能投产速度 > 需求增速 → 产能过剩开启（利空）
```

#### 指标2：库存水平（市场紧张度）
```
定义：产业库存分位数（vs 历史平均）

对于AI芯片：
- 芯片设计企业（汇顶）的库存：周转天数
- 代工企业（中芯）的库存：订单排队长度
- 下游（手机、服务器）的库存：历史分位数

监测方法：
- 通过财报披露的"存货周转率"反向推导
- 跟踪上市公司现金流表的库存变化
- 关注行业协会的库存指数发布

判断标准（库存分位数）：
- 0-20%  : 超低库存，供需最紧张 → 强势
- 20-50% : 低库存，供需紧张 → 乐观
- 50-80% : 正常库存，供需平衡 → 中性
- 80-100%: 高库存，供需开始失衡 → 警戒
```

#### 指标3：开工率（产能利用情况）
```
定义：实际产能使用 / 总可用产能

对于AI芯片：
- TSMC AI芯片开工率：当前 95%+ (超满负荷)
- 三星AI芯片开工率：80-90% (有余量)
- 中芯14nm开工率：90%+ (接近满载)

监测方法：
- 从芯片设计公司的财报中反向推导（订单饱和度）
- 关注代工企业的"产能利用率"披露
- 分析产品交期延后幅度

判断标准：
- 80%+持续 → 产能瓶颈形成，新产能投资启动信号 → 周期见顶预警
- 70-80% → 均衡状态
- <70% → 产能过剩，降价压力开始
```

#### 指标4：价格水平（绝对价格与成本比较）
```
定义：当前价格 / 成本线

对于AI芯片：
- 高端芯片代工价格：当前 120-130%(vs成本)
- 关键是"毛利率"而不是"绝对价格"

监测方法：
- TSMC & 三星的毛利率水平
- 中芯的毛利率水平
- 与历史平均比较

判断标准：
- 毛利率 > 历史平均+10% → 价格在高位，周期高处
- 毛利率 ≈ 历史平均 → 正常
- 毛利率 < 历史平均-10% → 底部机会
```

#### 指标5：采购/订单（需求前瞻指标）
```
定义：订单排期长度、交期延后幅度

对于AI芯片：
- TSMC AI订单排期：当前 6-9个月 (数据来源：行业报道)
- 客户采购增速：同比+50-100% (来源：我们Step3c的验证)
- 库存需求强度：库存周转加快 vs 历史平均

监测方法：
- 设计企业财报中"应付账款"的变化
- 代工企业的"订单储备"披露
- 行业媒体的交期延后报道

判断标准：
- 订单排期延长 & 交期后移 → 需求强劲
- 订单排期稳定 → 需求平衡
- 订单排期缩短 → 需求开始转弱（危险信号）
```

**输出格式：供需分析汇总表**
```
┌─────────────┬────────────┬─────────┬──────────────┬────────────┐
│   指标      │  当前值    │  分位数 │  历史中位    │  风险评级  │
├─────────────┼────────────┼─────────┼──────────────┼────────────┤
│ 产能增速    │  +20%/年   │   60%   │  +10%平均    │ 🟡 中等   │
│ 库存分位    │   45%      │   45%   │  50%         │ ✓ 正常   │
│ 开工率      │   92%      │   75%   │  80%         │ 🟡 偏高  │
│ 毛利率      │  48%       │   55%   │  45%         │ ✓ 正常   │
│ 订单排期    │  8个月     │   70%   │  6个月       │ 🟡 偏长  │
├─────────────┴────────────┴─────────┴──────────────┴────────────┤
│ 综合判断：当前处于"第2→第3阶段过渡期"，产能紧张但初现扩张信号  │
│ 风险：如果产能加速释放+订单排期缩短，3-6个月内可能转向第3阶段  │
└──────────────────────────────────────────────────────────────────┘
```

| 📱 订单延迟 | 交期排队时间缩短 | 提前1-2个月 | 🟡 中等 |
| 🏭 竞争加剧 | 新竞争对手大举进入 | 提前3-6个月 | 🟡 中等 |
| 💔 客户投诉 | 品质问题、交期问题增加 | 提前1个月 | 🟢 低 |

### 3.2 步骤3c改进：财报五维分析 + 周期PE调整

**修改估值维度：加入"周期PE"概念**

```markdown
## 3c.5 估值分析（改进）

### 改进前（传统PE/PB）
```
估值指标        汇顶      中芯      深南
PE-TTM          28        18        22

52周百分位      55%       51%       45%

```

### 改进后（加入周期PE概念）
```
1. 传统PE/PB分析（不变）
   PE-TTM          28        18        22
   PB              2.5       1.8       1.5
   52周百分位      55%       51%       45%

2. 周期调整分析（新增）
   
   当前EPS来源：从何而来？
   - 2024年利润中，多少来自"高利用率溢价"？(可能50-70%)
   - 多少来自"正常运营利润"？(可能30-50%)
   
   周期高峰预期EPS (假设产能满载+价格高位)：
   - 汇顶：当前EPS 2元 → 周期高峰预期EPS 3元 (利润空间50%)
   - 中芯：当前EPS 0.8元 → 周期高峰预期EPS 1.2元 (利润空间50%)
   - 深南：当前EPS 1.5元 → 周期高峰预期EPS 2.2元 (利润空间47%)
   
   周期PE计算（前瞻PE）：
   周期PE = 当前股价 / 周期高峰预期EPS
   - 汇顶：65元 / 3元 = 21.7倍 (比当前PE 28更便宜)
   - 中芯：25元 / 1.2元 = 20.8倍 (比当前PE 18稍贵)
   - 深南：15元 / 2.2元 = 6.8倍 (远便宜于当前PE 22)

3. 估值三角形
   
   维度1：定性极值（52周百分位）
   低估(0-30%)    正常(30-70%)    高估(70-100%)
     ✓               ✓ ← 当前         
   
   维度2：行业对标PE
   便宜(<15倍)    正常(15-25倍)    贵(>25倍)
               ✓ ← 当前位置      
   
   维度3：周期PE调整
   底部周期PE(<15) 正常周期PE(15-25) 顶部周期PE(>25)
     ✓ ← 当前位置    
   
   综合结论：三个维度都显示"相对便宜"，具备建仓价值

4. 估值四象限评价
   
   ┌─────────────────────────────────────┐
   │          估值便宜 + 周期早期          │ ← 汇顶、中芯、深南当前位置
   │    = 最佳投资区间                    │    评级：强烈买入
   ├─────────────────────────────────────┤
   │  估值便宜 + 周期末期  vs  估值贵 + 周期早期  │
   │   = 需要减轻          vs     保持      │
   ├─────────────────────────────────────┤
   │      估值贵 + 周期末期                │
   │   = 坚决规避                        │
   └─────────────────────────────────────┘
```

## 九、最后的话

### 本次改进的核心目标：

从 **"这家企业能赚钱吗？"** → 转向 **"这个商品周期处在哪个位置，我该怎么配置？"**

这意味着：
1. **研究商品优于研究企业** - 商品周期决定大方向
2. **时机优于标的** - 同样的股票，买入时机差一年，收益差5倍
3. **周期配置优于单一配置** - 不同周期阶段的配置应该不同
4. **定量锚点优于定性判断** - 用清晰的指标驱动决策，而不是感觉

### 应用顺序建议：

1. **立即**：用改进框架重新评估当前AI芯片配置 (1-2小时)
2. **Step4前**：建立库存、产能、交期的监测体系 (2-3小时)
3. **Step5前**：完成三个场景配置方案 + 锚点定义 (3-4小时)
4. **长期**：每周更新"决策执行表"，触发条件时快速反应 (30分钟/周)

### 预期效果：

- 更清晰的周期定位 ✓
- 更有纪律的进出场 ✓
- 更客观的风险管理 ✓
- 更高的投资回报率 ✓
