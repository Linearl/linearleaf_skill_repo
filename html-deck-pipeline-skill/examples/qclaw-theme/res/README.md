# qclaw-theme 本地资源说明

- 目录用途：为 `style-showcase-qclaw-theme.html` 提供离线可用的品牌图片资源。
- 来源站点：`https://qclaw.qq.com/`
- 资源列表：
  - `logo-nav.png`
  - `logo-combined.png`
  - `logo-cta.png`
  - `logo-footer.png`
  - `favicon.png`
  - `apple-touch-icon.png`

## 使用建议

- 优先使用相对路径（如 `./res/logo-nav.png`），避免演示现场受外网波动影响。
- 若对外发布材料，请先确认品牌素材使用权限。
