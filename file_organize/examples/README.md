# File Organize Skill Examples | 文件整理技能示例

## Overview | 概述

This directory contains examples demonstrating how to use the file-organize skill.

此目录包含展示如何使用文件整理技能的示例。

## Approach 1: Standard Template | 方式一：标准模板

### Example 1.1: Academic Research Template | 学术研究模板

```markdown
请参考工作流，整理D:\Downloads目录，使用模板A（学术研究型）。

目标：
- 将所有PDF论文按学科分类
- PPT课件按课程整理
- 研究资料按项目归档

约束：
- 不能删除任何文件
- 只允许移动和重命名
```

### Example 1.2: Business Office Template | 商务办公模板

```markdown
请整理E:\工作文档，使用模板B（商务办公型）。

要求：
- 当前项目文件放在显眼位置
- 历史项目归档
- 客户资料单独管理
```

### Example 1.3: Personal Life Template | 个人生活模板

```markdown
请整理我的桌面文件，使用模板C（个人生活型）。

目标：
- 照片按年月整理
- 重要文档分类保存
- 下载文件定期清理
```

## Approach 3: AI Custom Design | 方式三：AI定制设计

### Example 3.1: Creative Materials | 创作资料库

```markdown
请整理G:\我的创作资料库，我需要个性化的整理方案：

分类需求：
1. 按创作类型分类：小说、诗歌、剧本、散文
2. 每个类型下按状态分类：构思中、创作中、已完成、已发表
3. 素材文件单独管理：灵感记录、人物设定、背景资料

特殊要求：
- 手稿扫描件要保留原始文件名
- 参考资料按来源分类
- 协作文件单独放置

请生成详细的整理规则文档供我确认。
```

### Example 3.2: Multi-Role Worker | 多角色工作者

```markdown
我同时是程序员、摄影师和业余作家，需要整理我的综合资料库。

分类需求：
1. 编程项目：按语言和项目类型
```

### Example 3.3: Event-Based Organization | 事件导向整理

```markdown
我需要整理一次大型活动（公司年会）产生的所有文件。

文件类型：
- 策划文档
```

## Special Scenarios | 特殊场景

### Handling Duplicates | 处理重复文件

```markdown
请整理D:\照片备份，重点处理重复文件。

要求：
- 检测所有重复照片
- 同目录重复只保留最新的
- 跨目录重复保留所有（不同场景）
- 生成重复文件报告
```

### Thematic Folders | 主题文件夹

```markdown
请整理E:\下载，但注意保护主题文件夹。

说明：
- 有些文件夹是按项目/人名命名的
- 这些文件夹内的文件有关联性
- 不要拆散这些主题文件夹
- 整体移动到待确认目录
```

### Incremental Organization | 增量整理

```markdown
请只整理D:\工作\新文件目录中本周新增的文件。

要求：
- 只处理最近7天的文件
- 按现有目录结构分类
- 不改动历史文件
```

## Tips for Effective Organization | 高效整理技巧

1. **Backup First | 先备份**
   - Always create backup before organizing
   - Verify backup completeness

2. **Start Small | 从小处开始**
   - Test with a small folder first
   - Confirm the approach works before scaling

3. **Confirm Before Acting | 行动前确认**
   - Review AI's classification plan
   - Adjust rules as needed

4. **Handle Thematic Folders Carefully | 谨慎处理主题文件夹**
   - Don't break logical groupings
   - Move whole folders to 99_待确认

5. **Regular Maintenance | 定期维护**
   - Schedule weekly quick cleanups
   - Monthly deep organization
