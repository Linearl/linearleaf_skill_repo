## 💡 使用说明

### 新增Bug
1. 创建Bug详情文档: `to_fix/BUG-XXX_问题简述.md`
2. 在"待修复Bug"表格中添加记录
3. 更新统计数据

### 解决Bug
1. 将文档从 `to_fix/` 移动到 `fixed/`
2. 在Bug详情文档中填写解决信息
3. 将记录从"待修复"移至"已解决"
4. 更新统计数据

### Bug ID规则
- 格式: `BUG-XXX` (三位数字)
- 按顺序递增分配
- 已使用的ID不重复使用
